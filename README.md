# get-home-safe

## 작업환경 셋팅
1. Fork! 합니다. 🍴
2. Fork한 레포지토리를 적절한 위치에 클론 한 후 터미널에서 아래의 shell scrip를 실행시킵니다.

## 작업 시
1. 이슈 목록에서 할당된 업무의 이슈 번호를 찾습니다.
2. 이슈번호-작업내용 으로 새로 브랜치를 생성하고 작업합니다.
3. 작업이 완료된 내용을 Pull Request 합니다.
